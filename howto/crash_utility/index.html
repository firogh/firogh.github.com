Use crash utility to examine the running system

Mon Jan 1, 0001

  • howto

Prepare

Kernel pages info

Overview of Kernel Packages

Enbale debuginfo

/etc/yum.repos.d/fedora-updates.repo
/etc/yum.repos.d/fedora.repo
or
dnf –enablerepo=debug install kernel-debuginfo

locate vmlinux

rpm -ql kernel-debuginfo | grep vmlinux

Start to use crash

sudo crash /usr/lib/debug/lib/modules/4.18.9-200.fc28.x86_64/vmlinux
